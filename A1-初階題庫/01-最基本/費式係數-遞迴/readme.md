# 費氏數列

### Description
給你一個正整數 $n$，請問費氏數列的第 $n$ 項為何（請使用遞迴來解此題）。

費氏數列的計算公式如下：
$F_n = F_{n-1} + F_{n-2}$
$F_1 = 1$
$F_0 = 0$

### Hint
每一項都是前兩項相加

### Input
輸入只有一個正整數 $n$。
$1 \le n \le 20$

### Output
請輸出一個正整數，代表費氏數列的第 $n$ 項。
