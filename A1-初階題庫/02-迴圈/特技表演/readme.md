# 特技表演

## Description

有一個城堡有 n 個塔樓，樓高分別為 h1, h2, ..., hn，市長想要在城堡中心選一座高空技巧表演，該技巧表演會從某個大樓上飛向鄰近至地面。

為了表演人員的安全，清晰的視線穿高必須越來越低，請你找出一個最長的清晰路徑。

## Input

第一行有一個正整數 n (5 ≤ n < 100)。

第二行有 n 個正整數 h1, h2, ..., hn (1 ≤ hi ≤ 1000) 代表樓高。

- (60 分): n = 5
- (40 分): 無限制

## Output

輸出最長的清晰路徑長度。

## Sample Input 1

```
5
6 2 5 3 1
```

## Sample Output 1

```
3
```
