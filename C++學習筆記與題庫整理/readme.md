這是一份為初學者設計的 C++ 學習教材，包含語法教學與練習題庫。

## 使用方式

1. 打開任意 `.cpp` 檔案
2. 按 `Ctrl+Shift+B` 編譯執行
3. 查看 `output.txt` 結果
4. 試著修改程式碼，再執行看看！

## 學習建議

1. **先從 A0 語法介紹開始**，把每個主題都跑過一遍
2. **做 A1 初階題庫**，練習基本語法應用
3. **挑戰 A2 中階題庫**，學習演算法技巧
