# 差分陣列應用

## Description

給定一個長度為 n 的整數陣列 a，與 q 筆區間加法操作，每筆操作為 `l r x`，代表將區間 [l, r] 中每個數加上 x。

請輸出所有操作完成後的陣列內容。

## Input

- 第一行輸入兩個整數 `n`, `q`
- 第二行為陣列初始值，共 n 個整數
- 接下來 q 行，每行為 `l r x`

### 限制條件
- 1 ≤ n, q ≤ 100000
- -1000000 ≤ a[i] ≤ 1000000
- 0 ≤ l ≤ r < N
- -1000 ≤ x ≤ 1000

## Output

輸出最終的陣列內容，整數間以空格分隔。

## Sample Input

```
5 2
5 1 2 3 4
2 4 3
1 3 -2
```

## Sample Output

```
3 2 3 6 4
```

## Hint

請使用差分陣列技巧處理區間加法。

## 解題思路

**差分陣列**是前綴和的逆運算：
- 差分陣列 `diff[i] = a[i] - a[i-1]`
- 對區間 [l, r] 加上 x，只需要：`diff[l] += x` 和 `diff[r+1] -= x`
- 最後用前綴和還原：`res[i] = res[i-1] + diff[i]`

### 為什麼這樣做？

當我們對 `diff[l] += x`，做前綴和時，從位置 l 開始的所有數都會 +x。
但我們只想讓 [l, r] 這段 +x，所以要在 r+1 的位置 `-x` 抵消掉。

### 範例驗證

原始陣列：`a = [5, 1, 2, 3, 4]`（索引 1~5）

**Step 1：建立差分陣列**
```
diff[1] = a[1] = 5
diff[2] = a[2] - a[1] = 1 - 5 = -4
diff[3] = a[3] - a[2] = 2 - 1 = 1
diff[4] = a[4] - a[3] = 3 - 2 = 1
diff[5] = a[5] - a[4] = 4 - 3 = 1

diff = [0, 5, -4, 1, 1, 1, 0]  (索引 0~6)
```

**Step 2：處理操作**

操作 1：`[2, 4] +3`
```
diff[2] += 3  → diff[2] = -4 + 3 = -1
diff[5] -= 3  → diff[5] = 1 - 3 = -2

diff = [0, 5, -1, 1, 1, -2, 0]
```

操作 2：`[1, 3] -2`
```
diff[1] += (-2)  → diff[1] = 5 - 2 = 3
diff[4] -= (-2)  → diff[4] = 1 + 2 = 3

diff = [0, 3, -1, 1, 3, -2, 0]
```

**Step 3：用前綴和還原**
```
res[1] = diff[1] = 3
res[2] = res[1] + diff[2] = 3 + (-1) = 2
res[3] = res[2] + diff[3] = 2 + 1 = 3
res[4] = res[3] + diff[4] = 3 + 3 = 6
res[5] = res[4] + diff[5] = 6 + (-2) = 4

結果：[3, 2, 3, 6, 4] ✓
```

### 差分陣列 vs 暴力法

| 方法 | 每次操作 | q 次操作 |
|------|----------|----------|
| 暴力法 | O(n) | O(q × n) 太慢！ |
| 差分陣列 | O(1) | O(q + n) 超快！ |
