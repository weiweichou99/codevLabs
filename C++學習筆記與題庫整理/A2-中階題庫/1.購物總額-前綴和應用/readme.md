# 購物總額（前綴和的應用）

## Description

小悠最近在網路商城買了很多東西，她記錄了每天花費的金額。現在她想要知道，在某幾段期間內（例如第 2 天到第 5 天），她總共花了多少錢。由於查詢次數很多，她希望你能幫她快速完成這些計算！

給定一個長度為 N 的整數序列 A₁, A₂, ..., Aₙ，代表第 i 天花費的金額（可能為 0），以及 Q 筆查詢。每筆查詢給定兩個整數 L 和 R，請你輸出從第 L 天到第 R 天的總花費。

## Input

- 第一行輸入兩個整數 `N` 和 `Q`，分別代表天數與查詢數量
- 第二行輸入 `N` 個整數 A₁, A₂, ..., Aₙ，代表每天的花費
- 接下來 `Q` 行，每行輸入兩個整數 `L` 和 `R`，代表查詢的區間（1 ≤ L ≤ R ≤ N）

### 限制條件
- 1 ≤ N ≤ 10⁵
- 1 ≤ Q ≤ 10⁵
- 0 ≤ Aᵢ ≤ 10⁴

## Output

對於每筆查詢，輸出一行包含一個整數，代表該區間內的總花費。

## Sample Input

```
5 3
10 20 30 40 50
1 3
2 5
3 3
```

## Sample Output

```
60
140
30
```

## 解題思路

這題是**前綴和**的經典應用：
- 先建立前綴和陣列 `S[i] = A[1] + A[2] + ... + A[i]`
- 查詢區間 [L, R] 的總和 = `S[R] - S[L-1]`
- 注意：本題索引從 **1** 開始（1-based）

### 範例驗證

原始陣列：`A = [10, 20, 30, 40, 50]`

建立前綴和陣列（S[0] = 0 作為邊界）：
```
S[0] = 0
S[1] = 10
S[2] = 10 + 20 = 30
S[3] = 30 + 30 = 60
S[4] = 60 + 40 = 100
S[5] = 100 + 50 = 150
```

查詢計算：
| 查詢 | 公式 | 計算 | 答案 |
|------|------|------|------|
| [1, 3] | S[3] - S[0] | 60 - 0 | **60** ✓ |
| [2, 5] | S[5] - S[1] | 150 - 10 | **140** ✓ |
| [3, 3] | S[3] - S[2] | 60 - 30 | **30** ✓ |
