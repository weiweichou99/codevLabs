# 前綴和查詢練習

## Description

給定一個整數陣列與若干個查詢區間 [l, r]，請你回答每個區間內數字的總和。

本題可透過預先建立**前綴和陣列**來快速計算。

## Input

- 第一行包含兩個整數 `n`, `q`，表示陣列長度與查詢數量
- 第二行包含 `n` 個整數
- 接下來 `q` 行每行兩個整數 `l` `r` 表示查詢區間 [l, r]

### 限制條件
- 1 ≤ N, Q ≤ 100000
- -1000000 ≤ a[i] ≤ 1000000
- 1 ≤ L ≤ R ≤ N（索引從 1 開始）

## Output

輸出 q 行，每行為一個查詢區間內的總和

## Sample Input

```
5 2
1 3 -2 5 4
1 3
2 5
```

## Sample Output

```
2
10
```

## Hint

建議先計算前綴和陣列 `S[i] = a[1] + a[2] + ... + a[i]`

## 解題思路

**前綴和**是一種預處理技巧：
- 先算出從開頭到每個位置的累加和
- 查詢區間 [L, R] 的總和 = `S[R] - S[L-1]`
- 這樣每次查詢只需要 O(1) 時間！

### 範例驗證

原始陣列（1-based）：`a[1..5] = [1, 3, -2, 5, 4]`

建立前綴和陣列（S[0] = 0 作為邊界）：
```
S[0] = 0
S[1] = 1
S[2] = 1 + 3 = 4
S[3] = 4 + (-2) = 2
S[4] = 2 + 5 = 7
S[5] = 7 + 4 = 11
```

查詢計算：
| 查詢 | 公式 | 計算 | 答案 |
|------|------|------|------|
| [1, 3] | S[3] - S[0] | 2 - 0 | **2** ✓ |
| [2, 5] | S[5] - S[1] | 11 - 1 | **10** ✓ |
