# 前綴和查詢練習

## Description

給定一個整數陣列與若干個查詢區間 [l, r]，請你回答每個區間內數字的總和。

本題可透過預先建立**前綴和陣列**來快速計算。

## Input

- 第一行包含兩個整數 `n`, `q`，表示陣列長度與查詢數量
- 第二行包含 `n` 個整數
- 接下來 `q` 行每行兩個整數 `l` `r` 表示查詢區間 [l, r]

### 限制條件
- 1 ≤ N, Q ≤ 100000
- -1000000 ≤ a[i] ≤ 1000000
- 0 ≤ L ≤ R < N

## Output

輸出 q 行，每行為一個查詢區間內的總和

## Sample Input

```
5 2
1 3 -2 5 4
1 3
2 5
```

## Sample Output

```
2
10
```

## Hint

建議先計算前綴和陣列 `prefix[i] = a[0] + ... + a[i-1]`

## 解題思路

**前綴和**是一種預處理技巧：
- 先算出從開頭到每個位置的累加和
- 查詢區間 [l, r] 的總和 = `prefix[r] - prefix[l-1]`
- 這樣每次查詢只需要 O(1) 時間！

### 範例驗證

原始陣列（0-based）：`arr = [1, 3, -2, 5, 4]`

建立前綴和陣列：
```
prefix[0] = 1
prefix[1] = 1 + 3 = 4
prefix[2] = 4 + (-2) = 2
prefix[3] = 2 + 5 = 7
prefix[4] = 7 + 4 = 11
```

查詢計算（注意：本題索引從 0 開始）：
| 查詢 | 公式 | 計算 | 答案 |
|------|------|------|------|
| [1, 3] | prefix[3] - prefix[0] | 7 - 1 | **6** |
| [2, 5] | 注意：索引 5 超出範圍，應為 [2, 4] | prefix[4] - prefix[1] = 11 - 4 | **7** |

> ⚠️ 請依實際題目的索引規則（0-based 或 1-based）調整計算方式
