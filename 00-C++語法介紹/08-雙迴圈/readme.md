# 雙迴圈（巢狀迴圈）

## 什麼是雙迴圈？

迴圈裡面再放一個迴圈，用來處理**二維**的問題。

```cpp
for (int i = 0; i < 3; i++) {       // 外層：控制列
    for (int j = 0; j < 4; j++) {   // 內層：控制行
        cout << "(" << i << "," << j << ") ";
    }
    cout << endl;
}
```

輸出：
```
(0,0) (0,1) (0,2) (0,3)
(1,0) (1,1) (1,2) (1,3)
(2,0) (2,1) (2,2) (2,3)
```

## 經典應用

### 九九乘法表
```cpp
for (int i = 1; i <= 9; i++) {
    for (int j = 1; j <= 9; j++) {
        cout << i << "*" << j << "=" << i*j << "\t";
    }
    cout << endl;
}
```

### 印星星三角形
```cpp
for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= i; j++) {
        cout << "*";
    }
    cout << endl;
}
```

### 找所有配對
```cpp
for (int i = 0; i < n; i++) {
    for (int j = i + 1; j < n; j++) {
        // (i, j) 是一組配對
    }
}
```
