# 條件判斷 if / else

## 基本語法

```cpp
if (條件) {
    // 條件成立時執行
} else {
    // 條件不成立時執行
}
```

## 比較運算子

```cpp
a == b   // 等於
a != b   // 不等於
a > b    // 大於
a < b    // 小於
a >= b   // 大於等於
a <= b   // 小於等於
```

## 多重條件

```cpp
if (score >= 90) {
    cout << "A";
} else if (score >= 80) {
    cout << "B";
} else if (score >= 70) {
    cout << "C";
} else {
    cout << "不及格";
}
```
