# 迴圈 for / while

## for 迴圈

重複執行固定次數時使用。

```cpp
for (int i = 1; i <= 5; i++) {
    cout << i << endl;
}
// 印出 1 2 3 4 5
```

**拆解：**
- `int i = 1` → 起始值
- `i <= 5` → 繼續條件
- `i++` → 每次加 1

## while 迴圈

不確定次數時使用。

```cpp
int n = 5;
while (n > 0) {
    cout << n << endl;
    n--;
}
// 印出 5 4 3 2 1
```

## 常見用法

```cpp
// 計算 1+2+3+...+n
int sum = 0;
for (int i = 1; i <= n; i++) {
    sum = sum + i;
}
```

## 試試看

1. 印出 1 到 100
2. 計算 1+2+3+...+100 的總和
3. 印出九九乘法表的某一行
