# 變數與輸入輸出

## 什麼是變數？

變數就像一個「盒子」，可以存放資料。

```cpp
int age = 10;       // 整數（int）
double height = 1.5; // 小數（double）
string name = "小明"; // 文字（string）
```

## 輸入與輸出

```cpp
cin >> age;         // 從鍵盤讀取，存到 age
cout << age;        // 把 age 的值印出來
```

## 重點

- `int` = 整數（1, 2, 100, -5）
- `double` = 小數（3.14, 0.5）
- `string` = 文字（需要用雙引號 `""`）
- `cin >>` = 輸入
- `cout <<` = 輸出
