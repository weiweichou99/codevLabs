# 陣列

## 什麼是陣列？

陣列可以存放「多筆同類型的資料」。

```cpp
int scores[5];           // 宣告 5 個整數的陣列
int nums[3] = {1, 2, 3}; // 宣告並初始化
```

## 存取元素

陣列的索引從 **0** 開始！

```cpp
int a[3] = {10, 20, 30};

a[0]  // → 10（第一個）
a[1]  // → 20（第二個）
a[2]  // → 30（第三個）
```

## 搭配迴圈

```cpp
// 讀取 n 個數字
int n;
cin >> n;
int a[100];

for (int i = 0; i < n; i++) {
    cin >> a[i];
}

// 印出所有數字
for (int i = 0; i < n; i++) {
    cout << a[i] << " ";
}
```
