# 陣列

## 什麼是陣列？

陣列可以存放「多筆同類型的資料」。

```cpp
int scores[5];           // 宣告 5 個整數的陣列
int nums[3] = {1, 2, 3}; // 宣告並初始化
```

## 存取元素

陣列的索引從 **0** 開始！

```cpp
int a[3] = {10, 20, 30};

a[0]  // → 10（第一個）
a[1]  // → 20（第二個）
a[2]  // → 30（第三個）
```

## 陣列的限制（為什麼沒有 push_back？）

陣列是**固定大小**的，宣告時就決定了長度，無法動態增減。

| 操作 | 陣列 | vector |
|------|------|--------|
| 動態增加元素 | ❌ 不行 | ✅ `push_back()` |
| 動態刪除元素 | ❌ 不行 | ✅ `pop_back()` |
| 取得長度 | ❌ 要自己記 | ✅ `size()` |
| 大小可變 | ❌ 固定 | ✅ 可變 |

**結論**：需要動態增減元素時，請用 `vector`（見 07-vector）

## 陣列常用操作

```cpp
int a[5] = {3, 1, 4, 1, 5};

// 取得元素
a[0];          // 第一個元素 → 3
a[4];          // 最後一個元素 → 5

// 修改元素
a[2] = 100;    // 把第三個改成 100

// 用 sizeof 計算長度（僅限靜態陣列）
int len = sizeof(a) / sizeof(a[0]);  // → 5

// 填滿相同值
fill(a, a + 5, 0);      // 全部填 0
memset(a, 0, sizeof(a)); // 全部填 0（只適用於 0 或 -1）

// 複製陣列
int b[5];
copy(a, a + 5, b);      // 把 a 複製到 b

// 排序
sort(a, a + 5);         // 由小到大

// 反轉
reverse(a, a + 5);      // 反轉順序

// 找最大最小(內建函數)
int maxVal = *max_element(a, a + 5);
int minVal = *min_element(a, a + 5);
```

## 搭配迴圈

```cpp
// 讀取 n 個數字
int n;
cin >> n;
int a[100];

for (int i = 0; i < n; i++) {
    cin >> a[i];
}

// 印出所有數字
for (int i = 0; i < n; i++) {
    cout << a[i] << " ";
}
```
