# 二維陣列

## 什麼是二維陣列？

可以想像成一個**表格**，有列 (row) 和行 (column)。

```cpp
int a[3][4];  // 3 列 4 行的表格

// 初始化
int b[2][3] = {
    {1, 2, 3},
    {4, 5, 6}
};
```

## 存取元素

```cpp
a[0][0]  // 第 0 列、第 0 行
a[1][2]  // 第 1 列、第 2 行
```

## 搭配雙迴圈

```cpp
// 讀取
for (int i = 0; i < n; i++) {
    for (int j = 0; j < m; j++) {
        cin >> a[i][j];
    }
}

// 印出
for (int i = 0; i < n; i++) {
    for (int j = 0; j < m; j++) {
        cout << a[i][j] << " ";
    }
    cout << endl;
}
```

## 常見應用

- 棋盤、地圖
- 矩陣運算
- 圖片處理

## 試試看

1. 輸入一個 n×m 的矩陣，計算所有元素的總和
2. 印出矩陣的轉置（行列互換）
3. 找出矩陣中的最大值及其位置
