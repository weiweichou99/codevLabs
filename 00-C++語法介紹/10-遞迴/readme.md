# 遞迴 (Recursion)

## 什麼是遞迴？

函數**呼叫自己**，把大問題拆成小問題。

## 遞迴三要素

1. **終止條件**：什麼時候停下來
2. **遞迴呼叫**：呼叫自己處理更小的問題
3. **問題縮小**：每次呼叫問題要變小

## 經典範例：階乘

```cpp
// n! = n * (n-1)!
// 5! = 5 * 4! = 5 * 4 * 3! = ...

int factorial(int n) {
    if (n <= 1) return 1;    // 終止條件
    return n * factorial(n - 1);  // 遞迴呼叫
}
```

執行過程：
```
factorial(5)
= 5 * factorial(4)
= 5 * 4 * factorial(3)
= 5 * 4 * 3 * factorial(2)
= 5 * 4 * 3 * 2 * factorial(1)
= 5 * 4 * 3 * 2 * 1
= 120
```

## 經典範例：費氏數列

```cpp
// F(n) = F(n-1) + F(n-2)
// 1, 1, 2, 3, 5, 8, 13, 21, ...

int fib(int n) {
    if (n <= 2) return 1;
    return fib(n - 1) + fib(n - 2);
}
```

## 試試看

1. 用遞迴計算 1+2+3+...+n
2. 用遞迴計算 a 的 b 次方
3. 用遞迴印出 n 到 1 的倒數
