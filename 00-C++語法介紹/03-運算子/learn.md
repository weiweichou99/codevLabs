# 運算子

## 基本運算

```cpp
int a = 10, b = 3;

a + b   // 加法 → 13
a - b   // 減法 → 7
a * b   // 乘法 → 30
a / b   // 除法 → 3（整數除法，小數被捨去）
a % b   // 餘數 → 1（10 除以 3 餘 1）
```

## 重點

- `/` 整數除法會捨去小數
- `%` 取餘數，常用來判斷奇偶數

## 判斷奇偶數

```cpp
if (n % 2 == 0) {
    // n 是偶數
} else {
    // n 是奇數
}
```
