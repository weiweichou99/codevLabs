# 第一支程式：Hello World

## 程式碼說明

```cpp
#include <iostream>    // 引入輸入輸出功能
using namespace std;   // 使用標準命名空間

int main() {           // 程式從這裡開始
    cout << "Hello World" << endl;  // 印出文字
    return 0;          // 程式結束
}
```

## 重點

- `cout` = 輸出（印東西到螢幕）
- `<<` = 把右邊的東西送出去
- `endl` = 換行
- 每行結尾要加 `;`

## 試試看

1. 把 `Hello World` 改成你的名字
2. 多加幾行 `cout`，印出更多東西
