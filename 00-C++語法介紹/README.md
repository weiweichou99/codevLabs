# C++ 語法入門

這裡是 C++ 基礎語法的學習區，每個資料夾都是一個小主題。

## 學習順序

### 基礎篇
1. **01-第一支程式** - Hello World
2. **02-變數與輸入輸出** - 宣告變數、cin/cout
3. **03-運算子** - 加減乘除、餘數
4. **04-條件判斷** - if / else
5. **05-迴圈** - for / while
6. **06-陣列** - 存放多筆資料

### 進階篇
7. **07-vector動態陣列** - 可變長度的陣列
8. **08-雙迴圈** - 巢狀迴圈、九九乘法表
9. **09-函數** - 把程式碼包成可重複使用的單元
10. **10-遞迴** - 函數呼叫自己
11. **11-字串處理** - string 操作
12. **12-排序與搜尋** - sort、二分搜尋
13. **13-二維陣列** - 矩陣、表格

## 使用方式

1. 打開 `.cpp` 檔案
2. 按 `Ctrl+Shift+B` 編譯執行
3. 查看 `output.txt` 結果
4. **試著修改程式碼，再執行看看！**
